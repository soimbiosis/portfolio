Recommendation System

Assignment for 10-601: Machine Learning

The Assignment

The goal of this assignment was to use techniques described in class and in a paper we'd read to implement a recommendation system and train it on real data. We were provided with a dataset consisting of labeled training data and unlabeled training data, in this case, the MovieLens dataset.

Process

Read my design doc here. It provides a short summary of problem statement, description of data pre-processing and feature engineering, model implemented (including choice of parameters), and analysis of results.
