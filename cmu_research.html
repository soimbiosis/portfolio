Research with the Systems Biology Group

In Brief

The goal of this project was to start a user-friendly scRNA database, and develop a cell type classifier for novel sequence data. Single cell RNA sequencing is useful because it allows gene expression data to be examined at the cellular level. This helps understand things like cell function, mutations in small populations of cells, and evolution over time. Results of the project include scQuery, and the corresponding paper, A web server for comparative analysis of single-cell RNA-seq data.

Project Members

I worked with Dr. Matt Ruffalo (a doctoral student at the time), under the supervision of Dr. Ziv Bar-Joseph as part of the Systems Biology Group in the Computational Biology Department at Carnegie Mellon.

Project Contribution

I joined this project in an early, data collection stage. I worked on automated retrieval of single-cell RNA expression data from GEO (Gene Expression Omnibus) and extraction of source cell type information for the database development part of the project. I first generated and tested queries to download single-cell RNA expression data for mice from GEO. This was done using the Entrez API in Python. I then verified that the data was in fact mouse data, and extracted the cell type from a tag in the file. The result was a map per series (GSE) of gene expression samples (GSM) to cell types.
