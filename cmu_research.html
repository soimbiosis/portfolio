Research with the Systems Biology Group

In Brief

The goal of this project was automated retrieval of single-cell RNA expression data and extraction of source cell type information in order to start a user-friendly scRNA database, and develop a cell type classifier for novel sequence data. Results of the project include scQuery, and the corresponding paper, A web server for comparative analysis of single-cell RNA-seq data.

Project Members

I worked with Dr. Matt Ruffalo (a doctoral student at the time), under the supervision of Dr. Ziv Bar-Joseph as part of the Systems Biology Group in the Computational Biology Department at Carnegie Mellon.

Project Contribution

I worked on query development for data aggregation and extraction of cell type information for the database development part of this project. I first generated and tested queries to download single-cell RNA expression data for mice from GEO (Gene Ontology Online). This was done using the Entrez API in Python. I then verified that the data was in fact mouse data, and extracted the cell type from a tag in the file. The result was a map per series (GSE) of gene expression samples (GSM) to cell types
